Faça um algoritmo que receba o preço, o nome, a quantidade em estoque e o tipo (mercearia, limpeza e perecíveis) de produtos em 34 filiais de uma rede de lojas. Para encerrar a lista de produtos de uma filial, será inserido o valor -1 na variável de entrada referente ao preço.

Mostre, para cada filial, o valor total (em R$) em estoque. Mostre também a percentagem de produtos de mercearia, de limpeza e de perecíveis, contando toda as filiais. Por último mostre o preço do produto mais caro entre todos os produtos lidos e quantas filiais têm menos que R$ 60.000,00 em estoque.

Algoritmo
inteiro quantidade, contMercearia, contLimpeza, contPereciveis, contLoja, i, t;
literal tipo, nome;
real preco, valorTotal, maisCaro;

contMercearia = 0;
contLimpeza = 0;
contPereciveis = 0;
contLoja = 0;
maisCaro = 0;
para (i = 1; i <= 34; i++)
	valorTotal = 0;
	leia(preco);
	enquanto (preco != -1)
		leia (nome, quantidade, tipo);
		valorTotal = valorTotal + preco*quantidade;
		se (preco > maisCaro)
			maisCaro = preco;
		fimse;
		se (tipo == "mercearia")
			contMercearia = contMercearia + quantidade;
		senão
			se (tipo == "limpeza")
				contLimpeza = contLimpeza + quantidade;
			senão
				contPereciveis = contPereciveis + quantidade;
			fimse;
		fimse;
		leia(preco);
	fimenquanto;
	escreva (valorTotal);
	se (valorTotal < 60000)
		contLoja = contLoja +1;
	fimse;
fimpara;
t = contMercearia + contLimpeza + contPereciveis;
escreva (contMercearia/t, contLimpeza/t, contPereciveis/t);
escreva (maisCaro, contLoja);
fim.
